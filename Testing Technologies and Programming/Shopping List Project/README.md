# Shopping List App

## Project Overview

The Shopping List App is a simple web application that allows users to create and manage a list of shopping items. The goal of the project is to practice core front-end development skills including **HTML, CSS, JavaScript, and Git collaboration** while working as part of a development team.

The application will allow users to:

* Add items to a shopping list
* View the items they have added
* Remove items from the list
* Manage their list through a simple and easy-to-use interface

This project is being developed as a **static web application**, meaning it runs entirely in the browser without a backend server or database. The final version will be deployed using **GitHub Pages**.

## Purpose of the Project

The main purpose of this project is to help the team gain experience with:

* Collaborative development using **Git and GitHub**
* Working with **branches and pull requests**
* Structuring a small front-end web project
* Implementing basic **JavaScript functionality**
